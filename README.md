## 사용 방법

이 템플릿은 `npx`로 바로 실행하는 CLI 형태로 제공해요.
아래 순서대로 진행하면 쉽게 시작할 수 있어요.

---

## 프로젝트 만들기

아래 명령어를 실행하면 `my-app`이라는 이름의 폴더가 생기고,
그 안에 Square 프론트엔드 프로젝트를 만들어줘요.

```sh
npx create-square-app my-app
```

---

## 프로젝트 실행하기

프로젝트를 만들었다면, 생성된 폴더로 이동한 뒤 개발 서버를 실행해요.

```sh
cd my-app
pnpm run dev
```

---

## 필요한 환경

이 템플릿을 사용하려면 아래 환경이 필요해요.

* **Node.js 20 LTS 이상**
* **패키지 매니저: pnpm**

`pnpm`이 아직 없다면, 아래 명령어로 설치해요.

```sh
npm install -g pnpm
```
